Cara Integrasi API

      
Gunakan service OCR sebagai API multipart upload. Web cukup mengirim file dokumen dan membaca JSON hasil ekstraksi.

      

        Jangan taruh API key langsung di frontend publik. Kalau halaman ini dinaikkan ke GitHub/Vercel, simpan API key di environment variable serverless/API route, lalu frontend memanggil route proxy tersebut.
      

      

        
KTP lokal VPS
POST http://203.194.113.161/ocr/ktp

        
STNK lokal VPS
POST http://203.194.113.161/ocr/stnk

        
KTP agent VPS
POST http://203.194.113.161/ocr/agent/ktp

        
STNK agent VPS
POST http://203.194.113.161/ocr/agent/stnk

      

      
    
        
  • Method: POST
    • 
        
  • Body: multipart/form-data
    • 
        
  • Field file: file
    • 
        
  • Header auth: X-API-Key: <API_KEY>
    • 
        
  • Supported input: JPG, PNG, BMP, WEBP, PDF halaman pertama
    • 
      


      
Contoh cURL untuk IT

      
curl -X POST "http://203.194.113.161/ocr/ktp"   -H "X-API-Key: YOUR_API_KEY"   -F "file=@KTP.jpg"

curl -X POST "http://203.194.113.161/ocr/stnk"   -H "X-API-Key: YOUR_API_KEY"   -F "file=@STNK.pdf"


      
Contoh JavaScript dari Frontend ke Backend Proxy

      
const formData = new FormData();
formData.append("file", fileInput.files[0]);

const response = await fetch("/api/ocr/ktp", {
  method: "POST",
  body: formData
});

const result = await response.json();
if (result.input_assessment?.can_auto_publish) {
  // lanjut proses pembelian
} else {
  // tampilkan review / minta upload ulang
}


      
Contoh Vercel API Route Proxy

      
// app/api/ocr/ktp/route.js
export async function POST(request) {
  const body = await request.formData();
  const response = await fetch("http://203.194.113.161/ocr/ktp", {
    method: "POST",
    headers: {
      "X-API-Key": process.env.OCR_API_KEY
    },
    body
  });

  return Response.json(await response.json(), {
    status: response.status
  });
}
